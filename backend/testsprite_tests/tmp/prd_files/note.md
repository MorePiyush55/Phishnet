# PhishNet - Product Specification

## Overview
PhishNet is a phishing email detection platform that connects to a user's Gmail via Google OAuth and analyzes incoming emails for potential phishing indicators.

## Core Features
- Google OAuth 2.0 Login
- Fetch Gmail messages using Gmail API
- Analyze emails for phishing characteristics
- Display analysis results in React dashboard

## Tech Stack
- Backend: FastAPI
- Frontend: React.js
- Database: MongoDB Atlas (planned)
- Authentication: Google OAuth 2.0
